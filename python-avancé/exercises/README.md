### Fruit mix (à décliner avec "ensoleillement / eau / température", "azote / phosphore / potassium", ...

On leur fournit une fonction `taste(juice)`, juice étant un
dictionnaire de trois fruits/proportions :

    {"apple": 0.5, "banana": 0.25, "strawberry": 0.25}

La fonction taste renvoie un nombe entre 0 (j'aime pas) et 1 (j'adore).

Problème : donné une fonction `taste` opaque, trouver le mélange
parfait (qui renvoie ~1).
